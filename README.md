## Idea / Vision
A fast, affordable digital dystopian experience that helps young adults decompress. The user enters a session goal, chooses a scenery, and receives a guided escape experience.

## Flow / Features
1. Start
2. Display welcome message / video
3. User enters session goal
4. User chooses scenery
5. Prewritten AI - guided session script is displayed
6. End

## Tech stack
- Front-end: HTML, CSS, JS
- back-end ( when needed later): Python (Flask)
- Purpose: clickable MVPto validate user experience
